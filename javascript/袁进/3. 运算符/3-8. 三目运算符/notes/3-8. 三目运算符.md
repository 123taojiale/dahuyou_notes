## 3-8. 三目运算符

- 三目运算符
  - 书写方式： `表达式1 ? 表达式2 : 表达式3`
  - 运算步骤：
    - 对表达式1进行boolean判定
    - 如果判定结果为真，则执行表达式2，并返回表达式2的结果；否则，执行表达式3，并返回表达式3的结果。

---

- [x] 根据当前年份，返回这一年二月份的天数

```js
var year = 2000;
//是否是闰年
var isLeap = year % 4 === 0 && year % 100 !== 0 || year % 400 === 0;
//得到该年份二月的天数
var days = isLeap ? 29 : 28;
console.log("二月的天数为：" + days);
```

> 判闰年的逻辑：四年一闰，百年不闰，四百年一闰。

- [x] 练习

```js
var x = 1;
x = x++ >= 1 ? x++ * x++ : ++x * ++x;
console.log(x); // 6
```
