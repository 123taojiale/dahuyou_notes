## 3-6. 逻辑运算符

- 与（并且）
  - 符号：&&
  - 书写方式： 表达式1 && 表达式2
  - 运算步骤：
    - 将表达式1 进行 boolean 判定
    - 如果表达式1的判定结果为假，则直接返回表达式1，而不执行表达式2；否则，执行表达式2，并返回表达式2的结果。（短路规则）
- 或
  - 符号：||
  - 写法：表达式1 || 表达式2
  - 运算步骤：
    - 将表达式1 进行 boolean 判定
    - 如果表达式1为真，直接返回表达式1，不运行表达式2；否则，运行表达式2，并返回表达式运算结果；
- 非
  - 符号：!
  - 写法: !数据
    - 一元运算符
  - 运算步骤：
    - 将数据的 boolean 判定结果直接取反，非运算符一定返回 boolean 类型。
- 以下数据均判定为false, 其他数据全部为真
  - `null`
  - `undefined`
  - `false`
  - `NaN`
  - `''`
  - `0`

> 需要记住都有哪些数据会被判定为 false。
