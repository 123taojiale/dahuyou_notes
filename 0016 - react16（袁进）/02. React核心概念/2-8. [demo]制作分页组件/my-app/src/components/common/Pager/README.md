翻页组件

**属性**

| 属性名     | 数据类型 | 描述                                         |
| ---------- | -------- | -------------------------------------------- |
| size       | Number   | 每页展示多少条数据                           |
| total      | Number   | 一共有多少条数据                             |
| current    | Number   | 当前是第几页                                 |
| visibleNum | Number   | 用户可见多少个页码                           |
| onChange   | Function | current 发生变化时触发，参数为当前的最新页码 |
