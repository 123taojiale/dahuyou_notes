添加配置文件

如何向我们的工程中添加 ts 的配置文件：

  1. 在工程的根目录中创建一个 tsconfig.json 文件，该文件就是 ts 的配置文件。
  2. 在命令行，使用 tsc --init 命令来初始化一个 ts 的配置文件。

每一个配置项都有对应的注释，后续的学习过程中，需要用到的配置，会挨个介绍。

{
  "compilerOptions": {
    "target": "es2016",
    "module": "commonjs",
    "lib": ["es2016"],
    "outDir": "./dist"
  },
  "include": ["./src"]
}

compilerOptions：配置的是和编译相关的一些选项。

  • target：配置编译目标代码的版本标准；
  • module：配置编译目标使用的模块化标注；
  • lib：配置 ts 在默认情况下的运行环境，默认为 dom，它是一个数组，表示我们可以添加很多环境；
  • outDir
    • Specify an output folder for all emitted files.
