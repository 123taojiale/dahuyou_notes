# 题解1 暴力解法

```javascript
/**
 * 22-10-11
 * @param {string} s
 * @return {string}
 */
var reverseWords = function(s) {
  return s.split(' ')
    .map(item => item.split('').reverse().join(''))
    .join(' ');
};
```
