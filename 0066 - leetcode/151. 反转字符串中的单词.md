
## 题解1 暴力解法（正则 + 原生 api）

```javascript
/**
 * 22-09-28
 * @param {string} s
 * @return {string}
 */
var reverseWords = function(s) {
  return s.trim().replace(/\s+/g, ' ').split(' ').reverse().join(' ')
};
```
